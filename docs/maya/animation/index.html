Animation

List of all custom tools for animation.

Setup Shot

This tool connect to shotgrid and setup the length of the shot. You need to load a shot from ShotGun Tank to access this feature.

Attach Backpack to Joy

This tool connect the Backpack rig to Joy by using the ldRigNodes SpaceSwitches. Please select the reference of backpack and then of Joy before clicking.

Studio Library

This tool open the UI of Studio Library. This tool is external from IZES-Tools but is auto-detected.

Export Selection To Publish

Select the references groups to export, then click on the button and enter the version number (eg: 1; 2; 3; etc…). Then wait (this can be very long), every thing should be exported in the correct directory. Don’t forget to add them and not select the default geometry from Shotgrid publish tool.
